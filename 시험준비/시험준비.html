        CSS(Cascading Style Sheets)
        마크업 언어(HTML)가 실제로 화면에
        표시되는 방법(모양, 색상, 크기, 스타일...)을 기술하는 언어.
        -> HTML 코드가 화면에 어떤 스타일로 보여질지 지정하는 언어.
        W3C(웹 표준 지정 기관)에서 CSS를 표준으로 지정함.

        CSS Selector(선택자)
        현재 HTML 문서 내에 같은 태그를 선택하는 문자 또는 문자열.
        스타일을 적용하고자 하는 요소를 선택하는 방법.

        태그 선택자
        현재 HTML 문서 내에 같은 태그를 선택하는 선택자.
        태그명 {CSS 코드;}

        id 선택자(#id속성값)
        HTML 문서 내에 id 속성값이 일치하는 요소를 선택하는 선택자
        (중요!) id 속성값은 페이지 내에서 유일해야 한다!(중복값X!!)
        #id속성값 {css 코드;}

        class 선택자(.class 속성값)
        HTML 문서 내에 같은 class 속성 값을 가지는 태그를 모두 선택하는 선택자
        .class속성값 {css 코드;}
        (여러클래스 적용가능 class 속성값 순서 의미X style태그 순서에따라 다름)

        모든 요소 선택자(*)
        모든 요소를 선택하는 선택자
        * {css 코드;}

        기본 속성 선택자
        태그에 작성된 특정 속성을 선택하는 선택자
        선택자생략가능,속성값 양쪽 ""은 ''로 변경하거나 생략가능
        선택자[속성명="속성값"] {css 코드;}

        자식 선택자(>)
        지정된 요소 바로 하위에 존재하는 요소를 선택하는 선택자
        선택자1(부모) > 선택자2(자식) {css 코드;}

        후손(자손) 선택자( (띄어쓰기) )
        지정된 요소의 모든 하위에 존재하는 요소를 선택하는 선택자
        선택자1(조상) 선택자2(후손) {css 코드;}

        반응 선택자
        사용자의 움직임에 반응하여 스타일이 달라지는 선택자
        :active : 요소를 클릭하고 있는 경우
        :hover : 요소에 마우스가 올라가는 경우

        상태 선택자
        입력(input)양식의 상태에 따라 선택되는 선택자
        :focus 요소에 초점이 맞춰졌을때
        :checked 요소가 체크되었을때
        :enabled/ :disabled 요소가 사용 가능/불가능 상태일때

        동위 선택자
        동위관계(형제관계)에서 뒤에 위치한 요소를 선택하는 선택자
        A B B
        1) A 바로 뒤 B를 선택(+)
        A선택자 + B선택자 {css코드;}
        2) A 뒤에 있는 모든 B를 선택 (~)
        A선택자 ~ B선택자 {css코드;}

        문자열 속성 선택자
        속성 값의 문자열을 확인하여 스타일 적용하는 방식의 선택자

        1) 선택자[속성명 ~= "특정값"] { css 코드; }
        -> 띄어쓰기로 구분되어있는 여러 속성값이 작성된 속성 중
        속성값이 특정 값을 단어로 포함하는 요소를 선택

        2) 선택자[속성명 |= "특정값"] { css 코드; }
        -> 속성값이 특정값을 단어로 포함하는 요소를 선택
            단, "-" 기호로 구분, "-" 앞의 내용이 동일해야 한다.

        3) 선택자[속성명 ^= "특정값"] { css 코드; }
        -> 속성값이 특정값으로 시작하는 요소를 선택

        4) 선택자[속성명 $= "특정값"] { css 코드; }
        -> 속성값이 특정 값으로 끝나는 요소를 선택

        5) 선택자[속성명 *= "특정값"] { css 코드; }
        -> 속성값이 특정 값을 포함하는 요소를 선택

        일반 구조 선택자
        형제관계에 있는 요소 중 특정 요소를 선택하는 선택자
        (중요) 위치를 기준으로 구분함
        먼저 first,last,nth에 가서 요소확인후 맞으면 변경
        :first-child
        :last-child
        :nth-child(수열)
        :nth-last-child

        형태 구조 선택자
        선택된 형제 관계 요소 중 특정 요소를 선택하는 선택자
        (선택된 요소들을 기준으로 구분)
        타입의 몇번째인지로 확인
        :first-of-type - 같이 선택된 형제들 중에서 첫번째요소
        :last-of-type - 같이 선택된 형제들 중에서 마지막요소
        :nth-of-type(수열) - 같이 선택된 형제들 중에서 수열 번째 모든요소
        :nth-last-of-type(수열) - 같이 선택된 형제들 중에서 뒤에서 수열번째 모든요소

        부정 선택자
        괄호 내 선택자를 제외한 나머지 요소를 선택
        선택자1 > 선택자2:not(:nth-of-type()){css코드;}

        :only-child
        특정 요소의 자식이 하나밖에 없을 때
        :only-of-type
        특정 요소의 자식 중 지정된 형태의 자식요소가 하나만 있을때
        :empty
        자식요소가 없는 요소를 선택자(태그,글자,띄어쓰기 모두 포함)

        여러 선택자 동시 선택
        같은 스타일을 지정하고 싶으나 요소를 선택하는 선택자가 일치하지 않을때 사용
        선택자1, 선택자2, .... {css 코드;}

        CSS 선택자의 우선 순위로
        
        1) 동일 우선순위의 css 속성이 설정된 경우
            -> 제일 마지막 css 속성 반영
        2) 여러 선택자를 이용해서 같은 요소를 선택하더라도
            지정되는 css 속성이 다르면 모두 반영
        
        1순위 : css 속성 : 속성값 !important
        2순위 : inline-style 속성(요소에 직접 작성)
        3순위 : 아이디 선택자(#아이디 속성명)
        4순위 : 클래스 선택자(.class 속성명)
        5순위 : 태그 선택자(태그명)

        화면 배치 방법 : display 속성
        display: inline-block;
        block : 화면을 수직 분할(행)(width/height 사용O)

        none : 화면에 요소가 표시되지는 않으나 존재는 하고 있는 상태
        flex : 요소의 정렬되는 방향, 요소간의 간격을 유연하게 처리하는 형식

        HTML 요소
        1. content : 요소의 내용이 작성되는 영역
        2. padding : content와 border 사이
        (상하좌우, 상하/좌우, 상/좌우/하, 상/우/하/좌)
        3. border : 요소의 테두리
        border-width(상하좌우, 상하/좌우, 상/좌우/하, 상/우/하/좌)
        4. margin : 다른 요소와의 간격(겹치면 더큰곳)
        margin-auto : 좌우 요소간 간격을 자동으로 지정하여 가운데 정렬

        OOO-top / bottom / left / right

        box-sizing
        요소의 영역 중 실제 화면에 보여지는 content, padding, border
        는 각각 크기를 지정할수 있음(따로지정하다보니 전체적인 크기 파악,설정 힘듦)

        box-sizing: content-box(기본값): content영역만 크기 적용
        box-sizing: border-box : margin제외한 영역 크기 합산 적용

        float : 요소를 띄어서 좌/우로 정렬
        clear : float를 해제하는 속성(겹침문제 해결)

        position : 요소의 위치를 지정하는 속성
        relative;(상대적) : 지정된 요소 내부의 다른요소가 상대적인 위치를 지정할 수 있도록 기준이 되게 만듬
        absolute;(절대적) : 기본 요소의 배치 순서 무시 지정된 절대 위치에 요소를 배치
        fixed;(고정된) : 항상 고정된 위치에 요소를 배치

        Flexbox
        요소의 정렬 방향,순서,간격을 알아서 유연하게 처리해주는 방식
        1.구성(flex-container: 정렬할 요소를 감싸는 요소, item:정렬을 적용할 요소)
        2. 축(중심,교차,반대)

        container 전용속성
        flex-direction:메인축의 방향과 시작위치 지정
        row, column-reverse

        flex-wrap:내부 item들이 강제로 한줄에 배치되게 할것인지
                container가 줄어들면 여러줄로 배치할 것인지 지정
        nowrap:한줄 wrap:여러줄 wrap-reverse:여러줄뒤에서부터

        flex-flow : container의 메인축의 방향, 순서, wrap여부를 한번에 설정
        flex-flow : row-reverse wrap;

        justify-content: 메인축 방향으로 item의 정렬 방법 조정
        flex-start, flex-end, center, space-around, space-evenly, space-between

        align-items : 교차축 방향으로 item의 정렬 방법 조정
        flex-start, flex-end, center, baseline(content가 한줄에배치)

        align-content : 교차축 방향으로 item이 포장된(wrap)라인 정렬
        flex-start, flex-end, center, space-around, space-evenly, space-between

        item 전용속성
        order: item의 순서를 지정하는 속성
        flex-grow: 지정된 비율에 맞게 팽창함
        flex-shrink : 수축하는 정도를 지정
        flex-basis : 메인축 방향으로 크기 지정
        align-self : 교차축 방향으로 각각 정렬 지정
